## HTML 이란? ##
- 콘텐츠의 구조를 정의하는 마크업 언어이다.

- 여는 태그 <p> (Opening tag): 이것은 요소의 이름으로 구성되고 (여기에서는 p), 여닫는 꺾쇠괄호로 감싸집니다. 이것은 요소가 시작되는 곳, 또는 효과를 시작하는 곳

- 닫는 태그 </p> (Closing tag): 이것은 여는 태그와 같지만, 요소의 이름 앞에 전방향 슬래시가 포함된다는 점이 다릅니다.
